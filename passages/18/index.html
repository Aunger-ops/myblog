Hexo + leancloud 搭建国内快速独立博客

2020-04-16
Hexo

网上关于使用免费代码托管平台部署独立博客的文章很多：Github Pages、Netlify、七牛云OSS…..但大多数都是用LeanCloud部署评论存储，有点大材小用。正巧网站刚刚通过了备案，在此做一份使用LeanCloud来部署Hexo博客的教程。

优势

在下弃Netlify，投奔LeanCloud理由如下：

  1. 免费的HTTPS证书服务
  2. 免费的流量（有限）
  3. 国内更快的速度

  2. 填写 Github Repo仓库地址（如需使用私有仓库，则把下方的SSH keys填到Github处）

  1. 第一次部署测试

  1. Webhooks配置（使LeanCloud自动更新）
